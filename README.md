# GoldTrader - پلتفرم معاملات طلا و سکه

## درباره پروژه
پلتفرم حرفه‌ای معاملات آنلاین طلا و سکه با امکان خرید، فروش، مدیریت کیف پول و پنل ادمین کامل.

## اطلاعات ورود

### ادمین
- **شماره موبایل:** `09000000000`
- **رمز عبور:** `Admin@1234`

### کاربر تست
- **شماره موبایل:** `09123456789`
- **رمز عبور:** `User@1234`
- **موجودی:** ۱۰ میلیون ریال + دارایی‌های اولیه

## نصب سریع

```bash
# 1. نصب وابستگی‌ها
npm install

# 2. ساخت دیتابیس
node src/db/init.js

# 3. اجرا
npm start
# یا با PM2:
pm2 start ecosystem.config.cjs
```

سرور روی `http://localhost:3000` اجرا می‌شود.

## امکانات

### کاربری
- ورود با OTP یا رمز عبور
- معاملات طلا (۱۸ و ۲۴ عیار) و سکه (تمام، نیم، ربع)
- خرید/فروش بر اساس مقدار یا مبلغ
- کیف پول ریالی و دارایی
- درخواست برداشت وجه
- سیستم معرفی دوستان

### ادمینی
- داشبورد آماری
- مدیریت سفارشات و تأیید/رد
- مدیریت درخواست‌های برداشت
- قیمت‌گذاری دستی
- مدیریت کاربران
- گزارش روزانه/هفتگی/ماهانه

## استک فنی
- **Backend:** Node.js + Express.js 5
- **Database:** SQLite (better-sqlite3)
- **Auth:** JWT + OTP
- **Frontend:** Vanilla JS SPA
