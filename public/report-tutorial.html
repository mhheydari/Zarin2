نصب و راه‌اندازی
۱

نصب پیش‌نیازها

ابتدا Node.js نسخه ۱۸ یا بالاتر را نصب کنید.

node --version # باید v18+ باشد npm --version # باید 8+ باشد
۲

دریافت پروژه

git clone https://github.com/YOUR_USERNAME/goldtrader.git cd goldtrader
۳

نصب وابستگی‌ها

npm install
۴

ساخت دیتابیس

این دستور جداول، داده‌های اولیه، ادمین و کاربر تست را می‌سازد.

node src/db/init.js
۵

اجرای سرور

npm start # یا با PM2: pm2 start ecosystem.config.cjs

سرور روی پورت ۳۰۰۰ اجرا می‌شود: http://localhost:3000

۲
اطلاعات ورود پیش‌فرض

👑 ادمین سیستم

شماره موبایل:09000000000
رمز عبور:Admin@1234
دسترسی:مدیریت کامل سیستم

👤 کاربر تست

شماره موبایل:09123456789
رمز عبور:User@1234
موجودی ریالی:۱۰,۰۰۰,۰۰۰ ریال
دارایی:۵.۵ گرم هر نوع طلا + ۲ عدد سکه

همچنین می‌توانید با هر شماره موبایل دیگری ثبت‌نام کنید. در حالت development، کد OTP در response API برگشت داده می‌شود تا نیاز به SMS واقعی نباشد.

۳
ورود به سیستم
۱

ورود با OTP (کد تأیید)

تب "ورود با کد" را انتخاب کنید. شماره موبایل خود را وارد کنید. روی "ارسال کد تأیید" کلیک کنید.

۲

وارد کردن کد

کد ۶ رقمی را در خانه‌های مربوطه وارد کنید. در حالت development، کد در پیام سرور نمایش داده می‌شود.

۳

ورود با رمز عبور

برای کاربران از پیش تعریف‌شده، تب "ورود با رمز" را انتخاب کنید و شماره + رمز وارد کنید.

۴

استفاده از کد معرف

هنگام ورود با OTP، در صورت داشتن کد معرف، آن را در فیلد "کد معرف (اختیاری)" وارد کنید.

۴
انجام معامله (خرید و فروش)
۱

انتخاب دارایی

از بخش "معاملات" یکی از کارت‌های دارایی را انتخاب کنید (طلای ۱۸، ۲۴ عیار، سکه تمام، نیم سکه، ربع سکه).

۲

انتخاب نوع معامله

در پانل معامله، بین "خرید" (سبز) و "فروش" (قرمز) انتخاب کنید.

۳

انتخاب روش محاسبه

بر اساس مقدار: مثلاً ۲ گرم طلا بخرم.
بر اساس مبلغ: مثلاً با ۵ میلیون تومان طلا بخرم.

۴

وارد کردن مقدار

مقدار یا مبلغ را وارد کنید. سیستم به صورت خودکار معادل دیگر را محاسبه می‌کند.

۵

ثبت سفارش

روی "ثبت سفارش" کلیک کنید. اگر قیمت دستی باشد، معامله آنی تأیید می‌شود. اگر قیمت API باشد، سفارش در انتظار تأیید ادمین می‌ماند.

برای خرید باید موجودی ریالی کافی داشته باشید. برای فروش باید موجودی دارایی کافی داشته باشید. در صورت کمبود، پیام خطا نمایش داده می‌شود.

۵
کیف پول و دارایی‌ها
۱

مشاهده کیف پول

از منوی "کیف پول" می‌توانید موجودی ریالی، ارزش دارایی‌ها و ارزش کل پرتفو را ببینید.

۲

مشاهده دارایی‌ها

لیست تمام دارایی‌ها با موجودی و ارزش ریالی هر کدام نمایش داده می‌شود.

۳

شارژ کیف پول

شارژ کیف پول فقط توسط ادمین امکان‌پذیر است (از پنل ادمین ← مدیریت کاربران).

۶
درخواست برداشت وجه
۱

وارد کردن اطلاعات

از منو "برداشت وجه" را انتخاب کنید. مبلغ (حداقل ۵۰۰,۰۰۰ ریال)، شماره شبا (IR + ۲۴ رقم) و نام صاحب حساب را وارد کنید.

۲

ثبت درخواست

پس از ثبت، مبلغ از کیف پول شما رزرو می‌شود. درخواست در وضعیت "در انتظار" قرار می‌گیرد.

۳

فرآیند بررسی

کارشناسان با شما تماس می‌گیرند. پس از تأیید هویت، مبلغ به شبا واریز می‌شود. در صورت رد، مبلغ به کیف پول برمی‌گردد.

۷
سیستم معرفی دوستان
۱

مشاهده کد معرف

از منو "معرفی دوستان" کد منحصربه‌فرد خود را ببینید. این کد یکتا است و قابل تغییر نیست.

۲

اشتراک‌گذاری

با کلیک روی دکمه کپی، کد را کپی کنید. این کد را با دوستان و آشنایان خود به اشتراک بگذارید.

۳

ثبت‌نام با کد معرف

هنگام ثبت‌نام، دوستتان کد شما را در فیلد "کد معرف" وارد کند تا ثبت شود.

۸
پنل مدیریت (ادمین)
۱

داشبورد

آمار کلی: تعداد کاربران، سفارشات، حجم معاملات، سفارشات در انتظار. آخرین سفارشات و درخواست‌های برداشت.

۲

مدیریت سفارشات

مشاهده تمام سفارشات با فیلتر وضعیت. برای سفارشات در انتظار (API mode): دکمه تأیید ✅ یا رد ❌.

۳

مدیریت برداشت‌ها

مشاهده درخواست‌های برداشت. پس از تماس با مشتری و واریز وجه، دکمه تأیید را بزنید. در صورت رد، مبلغ به کاربر برمی‌گردد.

۴

قیمت‌گذاری

تغییر حالت قیمت (دستی/API). تنظیم درصد سود خرید و فروش. بروزرسانی قیمت هر دارایی به صورت جداگانه.

۵

گزارش‌گیری

مشاهده گزارش روزانه، هفتگی و ماهانه. آمار خرید و فروش به تفکیک دارایی. وضعیت درخواست‌های برداشت.

۹
اجرا با Docker
۱

build و اجرا

docker-compose up -d --build
۲

مشاهده لاگ‌ها

docker-compose logs -f goldtrader
۳

متوقف کردن

docker-compose down

فایل .env را قبل از اجرا با Docker تنظیم کنید. دیتابیس در پوشه data/ ذخیره می‌شود که در docker-compose به عنوان volume mount شده است.
