Ыұ. ШҜЫҢШӘШ§ШЁЫҢШі SQLite ШЁШ§ Ш¬ШҜШ§ЩҲЩ„ Ъ©Ш§Щ…Щ„ SQLite
Ыё Ш¬ШҜЩҲЩ„ Ш§ШөЩ„ЫҢ: users, assets, prices, orders, wallets, cash_wallets, withdrawals, referrals, notifications. ЩҮЩ…ЩҮ ШұЩҲШ§ШЁШ· ЩҲ Ш§ЫҢЩҶШҜЪ©ШівҖҢЩҮШ§ ШӘЩҶШёЫҢЩ… ШҙШҜЩҮвҖҢШ§ЩҶШҜ.
ЫІ. ШіЫҢШіШӘЩ… Ш§ШӯШұШ§ШІ ЩҮЩҲЫҢШӘ JWT
ЩҲШұЩҲШҜ ШЁШ§ OTP (Ъ©ШҜ Ы¶ ШұЩӮЩ…ЫҢ)ШҢ ЩҲШұЩҲШҜ ШЁШ§ ШұЩ…ШІ Ш№ШЁЩҲШұШҢ JWT token ШЁШ§ Ш§ЩҶЩӮШ¶Ш§ Ы· ШұЩҲШІЩҮШҢ middleware ШЁШұШ§ЫҢ admin ЩҲ userШҢ ШӯЩҒШ§ШёШӘ Ш§ШІ ШӘЩ…Ш§Щ… routeЩҮШ§.
Ыі. ШіЫҢШіШӘЩ… Щ…Ш№Ш§Щ…Щ„Ш§ШӘ Ъ©Ш§Щ…Щ„
Ш®ШұЫҢШҜ ЩҲ ЩҒШұЩҲШҙ Ыө ЩҶЩҲШ№ ШҜШ§ШұШ§ЫҢЫҢ (Ш·Щ„Ш§ЫҢ ЫұЫёШҢ ЫІЫҙ Ш№ЫҢШ§ШұШҢ ШіЪ©ЩҮ ШӘЩ…Ш§Щ…ШҢ ЩҶЫҢЩ… ШіЪ©ЩҮШҢ ШұШЁШ№ ШіЪ©ЩҮ). ШҜЩҲ ШӯШ§Щ„ШӘ: ШЁШұ Ш§ШіШ§Ші Щ…ЩӮШҜШ§Шұ ЫҢШ§ Щ…ШЁЩ„Шә. ШҜЩҲ ШӯШ§Щ„ШӘ ЩӮЫҢЩ…ШӘвҖҢЪҜШ°Ш§ШұЫҢ: ШҜШіШӘЫҢ (ШӘШЈЫҢЫҢШҜ ШўЩҶЫҢ) ЩҲ API (ЩҶЫҢШ§ШІ ШЁЩҮ ШӘШЈЫҢЫҢШҜ Ш§ШҜЩ…ЫҢЩҶ).
Ыҙ. ШіЫҢШіШӘЩ… Ъ©ЫҢЩҒ ЩҫЩҲЩ„ ЪҶЩҶШҜЪҜШ§ЩҶЩҮ
Ъ©ЫҢЩҒ ЩҫЩҲЩ„ ШұЫҢШ§Щ„ЫҢ Ш¬ШҜШ§ЪҜШ§ЩҶЩҮ ШЁШұШ§ЫҢ ЩҮШұ Ъ©Ш§ШұШЁШұ. Ъ©ЫҢЩҒ ЩҫЩҲЩ„ ШҜШ§ШұШ§ЫҢЫҢ ШЁШұШ§ЫҢ ЩҮШұ ЩҶЩҲШ№ Ш·Щ„Ш§/ШіЪ©ЩҮ. ШЁЩҮвҖҢШұЩҲШІШұШіШ§ЩҶЫҢ Ш®ЩҲШҜЪ©Ш§Шұ Щ…ЩҲШ¬ЩҲШҜЫҢ ЩҫШі Ш§ШІ ЩҮШұ Щ…Ш№Ш§Щ…Щ„ЩҮ. ЩҶЩ…Ш§ЫҢШҙ Ш§ШұШІШҙ Ъ©Щ„ ЩҫШұШӘЩҒЩҲ.
Ыө. ШіЫҢШіШӘЩ… ШЁШұШҜШ§ШҙШӘ ЩҲШ¬ЩҮ
Ш«ШЁШӘ ШҜШұШ®ЩҲШ§ШіШӘ ШЁШ§ ШҙЩ…Ш§ШұЩҮ ШҙШЁШ§ ЩҲ Щ…ШҙШ®ШөШ§ШӘ ШӯШіШ§ШЁ. Ш§Ш№ШӘШЁШ§ШұШіЩҶШ¬ЫҢ ШҙШЁШ§. ШұШІШұЩҲ Щ…ЩҲШ¬ЩҲШҜЫҢ ЩҮЩҶЪҜШ§Щ… Ш«ШЁШӘ ШҜШұШ®ЩҲШ§ШіШӘ. ШӘШЈЫҢЫҢШҜ/ШұШҜ Ш§ШҜЩ…ЫҢЩҶ ШЁШ§ ШЁШ§ШІЪҜШҙШӘ Ш®ЩҲШҜЪ©Ш§Шұ Щ…ШЁЩ„Шә ШҜШұ ШөЩҲШұШӘ ШұШҜ. ШіЫҢШіШӘЩ… Ш§Ш·Щ„Ш§Ш№ЫҢЩҮ.
Ы¶. ШіЫҢШіШӘЩ… Щ…Ш№ШұЩҒЫҢ (Referral)
Ъ©ШҜ Щ…Ш№ШұЩҒ ЫҢЪ©ШӘШ§ ШЁШұШ§ЫҢ ЩҮШұ Ъ©Ш§ШұШЁШұ. Ш«ШЁШӘ Щ…Ш№ШұЩҒЫҢвҖҢШҙШҜЪҜШ§ЩҶ ЩҮЩҶЪҜШ§Щ… Ш«ШЁШӘвҖҢЩҶШ§Щ…. Щ…ШҙШ§ЩҮШҜЩҮ Щ„ЫҢШіШӘ Ъ©ШіШ§ЩҶЫҢ Ъ©ЩҮ ШЁШ§ Ъ©ШҜ ШҙЩ…Ш§ Ш«ШЁШӘвҖҢЩҶШ§Щ… Ъ©ШұШҜЩҮвҖҢШ§ЩҶШҜ. Ъ©ЩҫЫҢ Ъ©ШҜ ШЁШ§ ЫҢЪ© Ъ©Щ„ЫҢЪ©.
Ы·. ЩҫЩҶЩ„ Щ…ШҜЫҢШұЫҢШӘ Ъ©Ш§Щ…Щ„
ШҜШ§ШҙШЁЩҲШұШҜ ШЁШ§ ШўЩ…Ш§Шұ Ъ©Щ„ЫҢ. Щ…ШҜЫҢШұЫҢШӘ Ъ©Ш§ШұШЁШұШ§ЩҶ ШЁШ§ Ш¬ШіШӘШ¬ЩҲ. ШӘШЈЫҢЫҢШҜ/ШұШҜ ШіЩҒШ§ШұШҙШ§ШӘ. Щ…ШҜЫҢШұЫҢШӘ ШҜШұШ®ЩҲШ§ШіШӘвҖҢЩҮШ§ЫҢ ШЁШұШҜШ§ШҙШӘ. ЩӮЫҢЩ…ШӘвҖҢЪҜШ°Ш§ШұЫҢ ШҜШіШӘЫҢ ШӘЩ…Ш§Щ… ШҜШ§ШұШ§ЫҢЫҢвҖҢЩҮШ§. ЪҜШІШ§ШұШҙвҖҢЩҮШ§ЫҢ ШұЩҲШІШ§ЩҶЩҮ/ЩҮЩҒШӘЪҜЫҢ/Щ…Ш§ЩҮШ§ЩҶЩҮ.
Ыё. ШіЫҢШіШӘЩ… Ш§Ш·Щ„Ш§Ш№ЫҢЩҮ
Ш§Ш·Щ„Ш§Ш№ЫҢЩҮ Ш®ЩҲШҜЪ©Ш§Шұ ШЁШұШ§ЫҢ ШӘШЈЫҢЫҢШҜ/ШұШҜ ШіЩҒШ§ШұШҙШҢ ШӘШЈЫҢЫҢШҜ/ШұШҜ ШЁШұШҜШ§ШҙШӘШҢ ШҙШ§ШұЪҳ Ъ©ЫҢЩҒ ЩҫЩҲЩ„. ЩҶШҙШ§ЩҶЪҜШұ ШӘШ№ШҜШ§ШҜ Ш®ЩҲШ§ЩҶШҜЩҮвҖҢЩҶШҙШҜЩҮ ШҜШұ header. Ш№Щ„Ш§Щ…ШӘвҖҢЪҜШ°Ш§ШұЫҢ ЩҮЩ…ЩҮ ШЁЩҮвҖҢШ№ЩҶЩҲШ§ЩҶ Ш®ЩҲШ§ЩҶШҜЩҮвҖҢШҙШҜЩҮ.
Ы№. ШұШ§ШЁШ· Ъ©Ш§ШұШЁШұЫҢ ШІЫҢШЁШ§ ЩҲ ШұЫҢШіЩҫШ§ЩҶШіЫҢЩҲ
Ш·ШұШ§ШӯЫҢ dark theme ШЁШ§ ШұЩҶЪҜ Ш·Щ„Ш§ЫҢЫҢ. Ъ©Ш§Щ…Щ„Ш§ЩӢ ШұЫҢШіЩҫШ§ЩҶШіЫҢЩҲ ШЁШұШ§ЫҢ Щ…ЩҲШЁШ§ЫҢЩ„ ЩҲ ШҜШіЪ©ШӘШ§Щҫ. Ш§ЩҶЫҢЩ…ЫҢШҙЩҶвҖҢЩҮШ§ЫҢ ЩҶШұЩ…. loading screen ЩҮЩҶЪҜШ§Щ… Ш§Ш¬ШұШ§. toast notifications. modal ШЁШұШ§ЫҢ ШӘШЈЫҢЫҢШҜЫҢЩҮвҖҢЩҮШ§.
ЫұЫ°. Docker ЩҲ Nginx
Dockerfile ШЁШұШ§ЫҢ containerize Ъ©ШұШҜЩҶ. docker-compose.yml ШЁШұШ§ЫҢ Ш§Ш¬ШұШ§ЫҢ ЫҢЪ©ЩҫШ§ШұЪҶЩҮ. nginx.conf ШЁШұШ§ЫҢ reverse proxy ЩҲ SSL. ШўЩ…Ш§ШҜЩҮ ШЁШұШ§ЫҢ deploy ШұЩҲЫҢ Ubuntu server.
